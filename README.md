# 📝 Python CLI To-Do List App

A simple command-line To-Do List application built with Python.  
It allows users to add, view, remove, and update tasks with persistent storage using a JSON file.

---

## 🚀 Features

- Add new tasks (name + description)
- View all tasks
- Remove tasks by index
- Update task status (Pending / Completed)
- Data stored permanently in `data.json`
- Colored terminal output
- Interactive menu system

---

## 🛠️ Technologies Used

- Python 3
- JSON (file storage)
- datetime
- termcolor
- pyfiglet
- inquirer

---

## 📂 Project Structure
Python To-Do List Code/
│
├── app.py # Main application file
├── data.json # Stores tasks automatically
└── README.md # Project documentation



⚙️ How It Works
Run the program
Choose an option from the menu
Add / view / update / delete tasks
All changes are automatically saved in data.json
